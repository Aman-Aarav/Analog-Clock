# Analog-Clock

Analog Clock using Qt (Shiny, Metallic & Reflective Design)

This project demonstrates custom painting, QPainter transformations, gradients, and animation using QTimer.

Features:

    Real-time analog clock with smooth second-hand motion
    Tick marks for hours and minutes with different thickness
    Custom gradient effects using QLinearGradient and QRadialGradient
    Automatic updates every second via QTimer

Key Classes & Concepts:

    QPainter:	Core class for 2D drawing
    QLinearGradient, QRadialGradient:	Create metallic, glossy, and glowing effects
    QTimer:	Triggers clock updates every second
    QTime:	Provides current system time
    QWidget::paintEvent:	Custom rendering function for the clock face
